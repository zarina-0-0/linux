# Задание 9: Изменение учетных записей

## Цель задания
Попрактиковаться в изменении пароля, имени пользователя и групп

## Ход работы

### Изменение параметров
```bash
usermod -l новое_имя старое_имя
groupmode -m новое_имя старое_имя
```

